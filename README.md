# TimeWarp Fest 2025 - College Festival Website

An immersive, time-traveling college festival website built with Next.js, Three.js, and Firebase.

## Features

- 🚀 Time Travel Theme with Dynamic UI
- 🔐 User Authentication System
- 📅 Interactive Event Registration
- 🗺️ Time Traveler's Map
- 📸 Interactive Photo Gallery
- 🎫 Ticketing System
- 🛍️ Merchandise Store
- 📱 Social Media Integration

## Tech Stack

- Next.js 14
- React 18
- Firebase
- Three.js
- GSAP
- Chakra UI
- Framer Motion
- TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/          # Next.js pages
├── styles/         # Global styles and theme
├── utils/          # Helper functions
└── public/         # Static assets
    └── images/     # Image assets
```
